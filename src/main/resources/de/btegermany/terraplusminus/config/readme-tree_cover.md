# Custom terrain elevation datasets

You can add json files to this directory to add custom tree cover datasets to TerraPlusMinus.

You can find the default tree cover dataset configuration file
[here](https://github.com/SmylerMC/terraminusminus/blob/master/src/main/resources/net/buildtheearth/terraminusminus/dataset/scalar/tree_cover.json5).

Your custom dataset will be loaded alongside the default ones.
In order to override them, make sure to give yours a higher priority.

For instructions on how to generate and configure your dataset,
you can refer to the [Terra++ wiki](https://github.com/BuildTheEarth/terraplusplus/wiki/Custom-terrain-dataset-guide).
